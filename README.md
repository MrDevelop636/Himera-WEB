## ⚙️ npm Scripts

- 🚀 **`npm run dev`** – Start development server
- 🏗️ **`npm run build`** – Build production version
- 👀 **`npm run preview`** – Preview the build
- 📝 **`npm run strapi`** – Start Strapi CMS server
